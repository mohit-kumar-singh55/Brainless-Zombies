# **Brainless Zombies（ブレインレス・ゾンビーズ）**

![ゲーム画像](./logo.png) <!-- ここに画像を追加 -->

Unity **6000.0.32f1** で開発した、素早い判断力と反射神経が試されるカラーアクションゲーム。
視点を瞬時に切り替え、正しい色の脳みそをゾンビにぶつけて人間に戻していくゲームです。

---

## 🎮 **ゲーム概要**

**Brainless Zombies** は、
**「前後に出現するゾンビに、同じ色の脳みそを投げて人間に戻す」** というシンプルでスピード感のあるゲームです。

特徴:

* ゾンビはプレイヤーの前後から近づいてくる
* ステージの照明が高速で点滅 → 位置把握が難しい
* 視点をパッと切り替えて状況を確認
* ゾンビが一定距離まで来ると **ゲームオーバー**

---

## 🧠 **基本ルール**

### ✔ 脳みそとゾンビの色

* **イエロー / マゼンタ / シアン**
* 同じ色を当てれば人間に戻る
* 違う色を当てると一瞬ハマってから落ちて消える

### ✔ 視点切り替え

* プレイヤーは **+Z（前）** か **–Z（後ろ）** のどちらかしか見られない
* スペースキーで即切り替え
* 点滅ライト中にゾンビを見るために必須

### ✔ ベルトコンベア

* 脳みそがプレイヤーへ流れてくる
* ゾンビはその1/3の速度
* ゾンビは出現から約 **8 秒** でプレイヤーへ到達

### ✔ 加速システム

人間に戻した数が増えるほどゲームが加速:

* 5体戻すごとに **速度 1.2倍**
* ゾンビも脳みそも早くなる

## 🕹 **操作方法**

### ✔ キーボード

| 動作        | キー          |
| --------- | ----------- |
| 前後の視点切り替え | **Space**   |
| 脳みそを右に弾く  | **→ 右矢印キー** |
| 脳みそを左に弾く  | **← 左矢印キー** |

### ✔ コントローラー

※コントローラー使えます！

---

## 💡 **特徴**

* ランダム高速点滅ライト（0.3〜1秒）
* 視点スナップで瞬時に前後を確認
* 一番手前の脳みそだけが射出される
* 色が違うと脳みそは無効
